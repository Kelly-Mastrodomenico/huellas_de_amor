# 📋 Introducción

## Descripción del proyecto

**Huellas de Amor** nace de la necesidad de digitalizar y centralizar el proceso de adopción de animales rescatados en España. Actualmente, muchas protectoras gestionan sus adopciones de forma manual o a través de redes sociales, lo que dificulta el seguimiento y la visibilidad de los animales disponibles.

Esta plataforma ofrece un espacio único donde protectoras, adoptantes y donantes pueden interactuar de forma organizada y transparente.

## Objetivos

### Objetivo general
Desarrollar una plataforma web completa (frontend + backend + base de datos) que facilite la adopción responsable de animales rescatados.

### Objetivos específicos
- Crear un catálogo de mascotas con sistema de filtros avanzados
- Implementar un sistema de solicitudes de adopción con seguimiento
- Desarrollar un programa de apadrinamiento con diferentes planes
- Integrar un sistema de donaciones con generación de certificados PDF
- Crear paneles diferenciados para usuarios y administradores
- Aplicar diseño responsive con técnica Mobile First
- Desplegar la aplicación en AWS

## Justificación

Cada año miles de animales son abandonados en España. Las protectoras necesitan herramientas digitales modernas para gestionar eficientemente los procesos de adopción. Huellas de Amor responde a esta necesidad con una solución tecnológica accesible y completa.

## Alcance

El proyecto cubre tres perfiles de usuario:

| Perfil | Capacidades |
|---|---|
| **Visitante** | Explorar catálogo, ver noticias, contactar |
| **Usuario registrado** | Solicitar adopciones, apadrinar, donar, descargar certificados |
| **Administrador** | Gestión completa, estadísticas, exportar informes |
